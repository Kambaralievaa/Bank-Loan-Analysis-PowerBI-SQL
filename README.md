# Bank_Loan_Analysis
Data analytics by building dashboards using real-world datasets.
In this project, I analyzed loan data to understand risk distribution and funding patterns.

🔍 Key insights:
86% of loans are performing well, while 13.8% show higher risk
Debt consolidation is the most common loan purpose
Loan performance varies across grades and home ownership types



🛠 Tools used:
Power BI | DAX | Power Query | Data Modeling
